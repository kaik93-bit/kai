Wie füge ich neue Testprojekte hinzu?

  • 1. Erstelle in deinem GitHub-Ordner einen neuen Unterordner (z. B. project1).
  • 2. Erstelle in diesem neuen Ordner eine index.html (das wird die Startseite des Testprojekts).
  • 3. Pushe die Änderungen mit deiner GitHub App. Das Projekt ist dann automatisch unter kaik93-bit.github.io/kai/project1/ erreichbar!
  • 💡 Tipp: Wenn du möchtest, dass auch deine Unterordner passwortgeschützt sind, kopiere einfach den Passwort-Teil aus dieser Datei in das Skript der neuen Seite.
