Dein Website-Link

Wenn du GitHub Pages in deinem Repository aktivierst, wird deine Website unter folgendem Link erreichbar sein:

https://kaik93-bit.github.io/kai/
So aktivierst du die Seite:
1
Pushe diese index.html in dein GitHub-Repository kaik93-bit/kai.
2
Gehe auf GitHub zu deinem Repository und klicke auf Settings (Einstellungen).
3
Wähle im linken Menü den Punkt Pages.
4
Wähle unter "Build and deployment" -> "Source" die Option Deploy from a branch. Selectiere deinen Branch (z. B. main oder master) und den Ordner / (root), und klicke auf Save.
